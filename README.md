# Live Server built using Vanilla JS and Node.js

The goal of this project is to build a live server that automatically reloads the browser when source files change. Currently, reloading works through polling. Future plan is to have the option to reload via WebSocket connection.

This project is still in the early stages of development. It is currently not using any third-party packages, and is only using vanilla JS and utilizing built-in modules of Node.js. 

The built-in modules of Node.js used so far:

(1) http <br/>
(2) fs <br/>
(3) path
